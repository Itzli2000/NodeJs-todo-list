# NodeJS Todo list

Todo list with NodeJs, to run in CLI.

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install the dependencies.

```bash
npm install
```

## Usage

Run the help command to see a full list of commands that the project accept.

```javascript
node index --help
```
